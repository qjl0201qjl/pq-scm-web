# PQ-SCM 新能源汽车感知质量与供应链协同决策支持系统

这是面向硕士论文答辩和课题展示的现代化 Web 前端原型。系统聚焦“评论分析 → 感知质量诊断 → QFD工程转化 → 潜在协同主体推荐”的管理决策链路，不展示模型训练细节。

## 运行

```bash
npm install
npm run dev
```

默认地址：

```text
http://127.0.0.1:5175
```

## 页面

- 首页总览
- 评论分析
- 感知质量诊断
- QFD工程转化
- 供应链协同
- 案例实证
- 报告中心

## 说明

第一版为纯前端 MVP，默认使用内置样例数据，也支持上传 CSV/Excel 评论数据进行展示级解析。术语统一采用“潜在协同主体”“协同改进对象”“协同方式建议”。
